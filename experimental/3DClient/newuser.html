  • Make sure to select the right IP and set the port to 2480 degrees or whatever you feel comfortable with.
  • Choose a Username which you will login with.
  • Choose a Password which you will keep secret with.
  • Choose a Character Name which you will use for every occasion.
  • Mix well with the Mix button to create your user.
